1. Clonar el proyecto desde GitHub a tu computadora.
2. Crear un entorno virtual para aislar las librerías del proyecto.
3. Instalar las librerías que están en requirements.txt.
4. Agregar tu clave de API en el archivo de configuración.
5. Probar que todo funciona antes de ejecutar el programa principal.
6. Ejecutar el programa principal para usar la IA de Gemini.
7. Esperar si hay errores por saturación del servicio, y volver a intentar.
